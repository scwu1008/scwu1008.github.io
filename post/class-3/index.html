机身与机头

2021-01-25

机身与机头

相机分类（大的分类）

  • 紧凑型（无法进行拆卸，添加外置部件）
  • 开放型（单反相机属于开放型）

单反相机由机身与镜头构成。

（数码相机分为全画幅相机和半/截画幅相机）

全画幅：相机的传感器（CMOS）的大小与胶片照相机的底片成像面一致

半/截画幅：CMOS相对于全画幅更小

相机机身组成

  • 卡口环（衔接机身和镜头）
  • 快门键（切换浏览模式与拍摄模式、半点用于对焦）
  • 快门（反光镜后方钢片—钢片式胶平面快门）
  • 手柄（一般为电池槽，用于稳定手持相机）
  • 热靴（接外置设备，例如：外置闪光灯、麦克风等其它非电子触点接触装置）
  • 取景器（光学取景器）
  • 主屏幕
  • 肩屏
  • 脚架螺口（装置脚架）
  • 曝光模式选择转轮（机身上方）

镜头参数说明（不同镜头参数标识符不同，类别相同）：

  • 变焦（Zoom：光学变焦；AF-S：宁静自动对焦，对焦速度更快）
  • 卡口类型（EF-S：佳能非全画幅相机卡口，DX：尼康非全画幅相机卡口）
  • 焦距段（18-35mm）
  • 最大光圈（1:3.5:5.6）
  • 图像稳定器（佳能镜头上为IS，尼康镜头上位VR）
  • 镜头口径（镜头前方横断面直径，前端开口）

焦距

「焦」：焦平面（机身上圆圈+横线）（传统/胶片照相机：底片所在位置；数码相机：CMOS所在位置）

「距」：镜头中心点到焦平面距离

焦距越长镜头「望远」能力越强，「视角」越窄，小物体所占画面比例更大。

135相机（胶片照相机）镜头视角和焦距的关系

以「焦距为标准」的镜头分类：

  • 按焦距是否可变：变焦镜头；定焦镜头

注：

景别：全景（从头到脚）、中景（半身）、近景（胸部以上）、脸部（特写）、眼睛（大特写）、远景（画面中人物只有一个点，大面积为环境）

变焦镜头特点：在同一地点能够实现不同景别

定焦镜头特点：通过摄影师位置移动，实现不同景别

  • 按焦距长度：光脚；标准；中焦；长焦

注：

（「焦距短的镜头」拍摄「小景别」会导致画面变形，「夸大近大远小特点」）

（焦距越长，背景越虚，相对广角丧失空间感）

全画幅相机焦距分类：

广角：28mm以下；广阔的视角特性；画面变形更为显著的特性；夸大平面画面透视关系，立体表现更加明显。

标准镜头（人单眼所看到的视角，约45度）：50mm左右；画面非常自然的特性（由于接近人眼视角）；

中焦镜头（肖像镜头—中近景）：85-135mm左右；背景朦胧化，变虚；

长焦镜头（望远/摄远镜头）：大于135mm；远处拍摄的特点（野外生物，运动员）；背景虚化的特点；

非全画幅：广角（尼康：28mm/1.5，佳能：28mm/1.6），标准镜头（类似广角计算），其它类似。

焦距、等效焦距、视角、透视的理解

  • 按特殊用途：微距镜头；移轴镜头等

注：

（每个镜头都有其「工作距离」，所以距离太近则会导致「无法对焦」）

微距镜头特性：距离近仍能看清拍摄主体。「拍摄物体面积」和「CMOS成像面积」达到1:1放大率。

移轴镜头特性：扭曲镜头前端，实现「前端与后端镜片不在同一平面上」，呈现一定立体关系；用于建筑摄影中校正透视关系（消除近大远小带来建筑在画面中倾斜/跌倒的影响）

「光圈」的说明：

光圈是一个用来「控制光线透过镜头，进入机身内感光面的光亮」的装置，是在镜头内。

（光圈值越大，后端开口/光圈大小越小，进光越少；值越小，后端开口/光圈大小越大，进光越多。「光圈值可以理解为前端开口和后端开口大小的比例，便于记忆」）

注：

最大光圈（1:3.5:5.6）：最大光圈（最小光圈值）在3.5-5.6之中变化，表明其为「非恒定光圈镜头」（随着焦距变化，镜头最大光圈会发生改变），随着焦距变大，最大光圈会缩小（光圈值增大）。

（定焦镜头没有光圈值的浮动，即为「恒定光圈」；但变焦镜头也存在恒定光圈）

自动对焦

一般在镜头会有自动对焦锁，用于切换「自动」和「手动」对焦方式。

自动对焦：半点快门时，「相机主动通过移动内部的对焦镜片」，使得影响从模糊变清晰。

手动对焦：手动转动对焦环，使得影像变清晰。

注：

非常暗的环境/拍摄没有层次的物体（例如拍摄白纸等）/透过玻璃拍摄外部物体等「自动对焦无法生效」。

镜头上标识为A/M：表明相机进入自动对焦状态，仍然可以拧动对焦环，加强对焦精度。

防抖

不同品牌镜头上有不同的标识符。
